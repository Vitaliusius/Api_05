# Поиск и сравнение вакансий программистов

Данный скрипт запрашивает вакансии с двух порталов Headhunter и SuperJob по языкам программирования указаным пользователем.

Для каждой вакансии высчитывается примерна зарплата. На основании этих данных, считается средняя зарплата по указаным языкам программирования проверяя все вакансии.

В конце выводит таблицу для каждой платформы, в которой указаны языки программирования, количество вакансия найденых для каждого языка, количество вакансий которое было учтено при подсчете средней зарплаты и саму среднюю зарполату.

## Зависимости

Python3 должен быть уже установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

## Как запустить

Для запуска скрипта необходимо создать в корневом каталоге файл '.env'. В котором указать интересующие языки программирования как в примере ниже, присвоив их переменной LANGUAGES:

```
LANGUAGES = Python, Java, Ruby, Go
```
Так же для работы скрипта необходим секретный ключ SuperJob, получить его можно зарегистрировав приложение [здесь](https://api.superjob.ru/). 

Полученный ключ присвоить переменной SECRET_KEY_SJ в файле '.env', как в примере ниже:

```
SECRET_KEY_SJ=sdfhjgsjdjh5567gjkkhggh6567823060283
```

Для запуска скрипта использовать команду:

```
python main.py
```
